## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or setup required
- Internet connection for CDN resources

### How to Use

1. **Open the Website**
   - Navigate to `index.html` in your web browser
   - Or set up a local server for better performance

2. **Browse**
   - Explore the homepage and feature highlights
   - View success stories and testimonials

3. **Search for Matches**
   - Fill out the search form on the hero section
   - Select gender, looking for, age, and city preferences
   - Click "Let's Begin" to start searching

4. **Login**
   - Click the "Login" button in the top navigation
   - Enter credentials or choose OTP login
   - New users can sign up for free

5. **Create Profile**
   - Register for free
   - Set up your matrimony profile
   - Verify your profile details
   - Start connecting with matches
